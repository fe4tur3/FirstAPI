# FirstAPI
Hatte ich als kurzen Übungsauftrag. 
Aufgabe war einen cmd-Server anzusprechen mit einem Button, um eine Ausgabe zu erhalten.
Ein zweiter Button sollte selbst eingegebenen Text einfach verarbeitet ausgeben. 
JavaScript der Einfachheit halber verwendet. 

Fokus legte ich auf die Optimierung meines Arbeitsweges: Sicherheit, DRY-Prinzip anwenden etc.
